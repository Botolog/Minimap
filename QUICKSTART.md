# Quick Start Guide

## Get Running in 3 Steps

### 1. Open the App
- Double-click `index.html` or open it in Chrome/Firefox
- Or run a simple web server:
  ```bash
  cd /home/botolog/Documents/Car
  python3 -m http.server 8000
  ```
  Then visit: `http://localhost:8000`

### 2. Grant Permissions
- Click "Allow" when browser asks for location access
- Wait 10-30 seconds for GPS to lock

### 3. Start Driving!
- Map will center on your location
- Map rotates as you turn (car stays centered)
- Speed updates in real-time

## Troubleshooting

**GPS not working?**
- Grant location permissions
- Enable device GPS
- Go outdoors for better signal

**Map not loading?**
- Check internet connection
- Try refreshing page
- Cache tiles for offline use

**Rotation laggy?**
- Close other apps
- Try disabling trail history
- Increase GPS interval
